# 🍽️ Parampara & Palms – Food Ordering System

A full-stack **Food Ordering Web Application** built with **React.js** and **Django REST Framework**. The application allows users to browse menu items, add products to a cart, place orders, choose payment options, track order status, and download invoices. It also includes an **admin dashboard** for managing food items and customer orders.

🔗 **Live Demo:** https://parampara-and-palms-1.onrender.com

---

## 📌 Features

### 👤 Customer Features
- User registration and login
- Browse food categories and menu items
- Search and filter food items
- Add/remove items from the cart
- Place orders
- Choose payment options
- Track order status
- Download invoices

### 🛠️ Admin Features
- Add, update, and delete food items
- Manage categories
- View and manage customer orders
- Update order status
